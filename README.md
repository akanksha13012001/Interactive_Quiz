# Interactive Quiz Website

An interactive web-based quiz application built using HTML, CSS, and JavaScript. Users can answer multiple-choice questions, get instant feedback, and see their scores at the end of the quiz.

## 🚀 Features

- Dynamic multiple-choice quiz
- Real-time score tracking
- Feedback for correct/incorrect answers
- Responsive design for all screen sizes
- Restart quiz functionality

## 🛠️ Technologies Used

- **HTML5** – Structure and content
- **CSS3** – Styling and layout
- **JavaScript (Vanilla)** – Logic and interactivity
